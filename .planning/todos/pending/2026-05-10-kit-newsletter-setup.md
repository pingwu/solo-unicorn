## Problem

Need to import email list to Kit (ConvertKit) and establish newsletter workflow for class announcements and technology updates. Currently have CSV list ready; want to start with manual/semi-automatic sending, then transition to API automation eventually.

## Solution

**Phase 1 (Manual/Semi-auto):**
1. Import CSV subscriber list to Kit
2. Create two newsletter templates: class announcements and tech updates
3. Set up manual sending + scheduled sends
4. Use tags/segments for organization

**Phase 2 (API Automation - later):**
1. Integrate Kit API for triggered sends
2. Explore Zapier/Make for conditional automation
3. Document automation workflows
