# Contributing to Solo Unicorn Builder

Thanks for your interest in contributing. Whether you're fixing a typo, adding a skill, or improving documentation, every contribution helps builders transition into AI-native development roles.

## Quick Start

1. Fork the repo and clone it locally
2. Run the init setup with your AI coding agent: **"Run the init unicorn setup"**
3. Create a branch, make your changes, and open a pull request

## Full Guide

See **[docs/contributing.md](docs/contributing.md)** for the complete step-by-step guide, including how to create and contribute new skills.

## Types of Contributions

- **New skills** — Package your domain expertise as a skill in `skills/`
- **Bug fixes** — Fix issues in documentation or code
- **Documentation** — Improve guides, fix typos, add examples
- **Success stories** — Share what you built with Solo Unicorn Builder in [GitHub Discussions](https://github.com/pingwu/solo-unicorn/discussions)

## Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold it.

## Questions?

Open a [GitHub Discussion](https://github.com/pingwu/solo-unicorn/discussions) or reach out at [ask@ping-ai.com](mailto:ask@ping-ai.com).
